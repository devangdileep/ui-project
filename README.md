# Wanderly Tours

A simple React tours and travels website with Supabase backend support.

## Features

- Customer and travel agent login
- Travel package listing
- Customer package requests
- Agent package add, edit, and delete
- Agent request accept and decline workflow
- Responsive design

## Setup

```bash
npm install
npm run dev
```

Create `.env.local` from `.env.example`, then run `supabase/schema.sql` in Supabase.
